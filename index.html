Graduate Paper on "Finding an effective and suitable approach for Mobile Application Development"

  • Abstract: Rapid improvements in mobile application devel- opment technologies and the expansion of high-quality, large- scale wireless networks have given rise to new applications and complexities in computer science and business domains. Software development for such contexts now faces additional difficulties as a result of the particular needs and limits associated with mobile systems, which necessitate significant improvements to traditional application development approaches. Software Development Lifecycle (SDLC) is crucial in Desktop or web application development. Mobile applications are being devel- oped for deployment in smart phones. Looking at the rising need of mobile applications and the associated development complexity, it is imperative to have a dedicated framework for mobile application - Mobile Application Development Lifecy- cle (MADLC). Identification, Design, Development, Prototyping, Testing, Deployment, and Maintenance are the proposed phases that make up this lifecycle. This lifecycle tackles some of the distinct constraints of mobile applications, such as life span, complex functionalities, fewer physical interfaces, more number of screens for user interaction, battery and memory usage, cross platform development and maintenance. By adhering to this proposed development approach developers can develop reliable and effective control applications. This proposed development approach also intended to help developers to opt for an effective and suitable development approach based on their software requirement deliverables.
